新疆自治区党委教育工委到我校调研交流基层党组织建设情况

部门：党委组织部 供稿：高兰芳 审核：孙景仙 齐勇 发布时间：2017-05-12
阅读次数： 【字号：大 中 小】

　　5月10日，新疆自治区党委教育工委常务副书记王志军等一行4人，在北京市委教育工委组织处副处长赵学智陪同下到我校调研交流基层党组织建设情况。党委书记王建中、校长张爱林、纪委书记何志洪在大兴校区四合院会议室热情接待了来访嘉宾，相关部门负责人及部分二级学院党委书记参加调研交流会，会议由张爱林主持。

 

 

　　王建中代表全校师生对王志军副书记一行的来访表示热烈欢迎，并简要介绍了我校的历史沿革和发展现状。王建中指出，对口支援新疆工作是中央的一项重大战略部署，为新疆培养社会主义建设的合格建设者和可靠接班人是学校落实国家战略、支持新疆发展的重要使命，学校将继续按照中央和市委的部署，加大对少数民族学生的培养力度，发挥学科专业优势，做好对口支援学校建设工作，积极参与新疆建设，为新疆发展做出新的贡献。

 

 

　　王志军对我校的热情接待表示衷心感谢，他简要介绍了新疆自治区党委教育工委的成立、机构设置、任务分工及职能发挥等，希望向北京教工委及高校学习党建和思想政治工作方面先进经验，更好发挥指导和管理作用，全面推动重点工作。

 

 

　　随后，与会人员共同观看了我校宣传片，党委组织部部长孙景仙作了题为《深入落实党建工作责任制，以党建工作成效推动学校事业改革发展》的汇报，从落实主体责任、层层传导压力、推动全面从严治党向纵深发展，坚持全面从严、持续转变作风、不断加强干部队伍建设，发挥党建优势、服务中心工作、促进学校事业快速发展等三方面介绍了学校党建工作情况。

　　与会人员围绕高校院系党组织建设制度和作用发挥、院系党组织换届的经验做法、党组织活动开展、党务干部设置、党建特色工作经验等主题进行了深入交流。会后，来访嘉宾参观考察了我校测绘学院、校史馆和正在我校举办的“一带一路”历史建筑摄影・手绘艺术展。
