“一带一路”高峰论坛校园安保维稳工作

部门：保卫部（处） 供稿：胡锐 摄影：汪洋海容 审核：牛磊 白 发布时间：2017-05-11
阅读次数：193 【字号：大 中 小】

　　5月10日，北京市委教工委委员、干部处处长陈江华一行赴大兴校区，就“一带一路”高峰论坛期间校园及周边安全工作开展专项督查，副校长李维平带领相关职能部门负责人参加汇报会，并陪同视察。

 

 

　　下午两点，陈江华一行首先在大兴校区四合院会议室听取了学校“一带一路”高峰论坛期间的专项工作汇报，保卫处处长牛磊代表学校从三个层面、十项举措详细介绍了学校近期的安全稳定工作，党政办主任白、学工部部长黄尚荣、资后处副处长罗会文分别就学校网络安全工作、重点人群教育管理、食堂食品安全管控等进行了补充汇报。副校长李维平在总结汇报中指出，学校领导高度重视“一带一路”高峰论坛的校园安保维稳工作，一是学校高度重视，党委常委会、校长办公会多次专题讨论近期的安稳工作；二是严格落实上级文件精神，层层分解主体责任；三是将近期工作纳入“平安校园”建设整体工程，通过上下联动、点面结合、统筹协同等全面确保高峰论坛期间校园及周边安全。

　　教工委委员、干部处处长陈江华在听取学校的安全工作汇报之后，从三个方面积极肯定了我校的安稳工作：一是整体部署全面，从学校领导重视，到多部门联合开展工作，再到广大师生的安全教育，对高峰论坛期间安全稳定的意义认识到位，并开展了多方面多层次的有效工作；二是细处保障有力，学校在安稳工作的细节上有想法有创新有措施，如落实安全责任制的实验室三级管理制度、校园周边治理的全面联动及飞行器的有效管控等；三是长效机制健全，学校安全稳定工作相关的制度建设完备，覆盖问题全面。陈江华就学校接下来的工作提出了进一步要求，一是进一步落实安全责任制，层层传导压力；二是要在制度健全的前提下继续狠抓落实，及时发现并整改问题，从源头上管控风险；三是做好两校区校园安稳工作人财物的合理分配，确保学校全面安全稳定。

 

 

　　汇报会之后，陈江华一行在李维平及相关职能部门负责人的陪同下，先后实地考察了大兴校区校园监控指挥中心运行情况及北大门管控情况，并考察了重点部位学校图书馆的安全管理情况，督查组就学校技防建设、特殊时期校园管控、重点部位安防等展开进一步督查与指导。
