房山区领导与中铁二十二局集团有限公司领导进行座谈

来源：区信息中心 | 作者：刘婷 | 发布日期：2016-04-23 天气 阴 南风 气温 22-44度

　　5月10日，房山区领导与中铁二十二局集团有限公司领导进行座谈，区委副书记、区长陈清，中铁二十二局集团有限公司董事长、党委书记侯希承出席会议。
　　作为国家特大型建筑施工企业的中铁二十二局集团有限公司，具有铁路工程施工总承包特级资质，公路工程、房屋建筑、水利水电、市政工程施工总承包壹级资质，土石方、桥梁、隧道、钢结构工程专业承包壹级资质，城市轨道交通工程专业承包资质。具有对外承包工程经营资格。经营领域涉及工程建设、房地产开发、机械设备营销租赁、铁路运输等。
　　侯希承表示，房山具有独特的地理优势和区位优势，合作潜力和空间较大。希望与房山区进一步加强交流，在公司迁址、棚户区改造、特色小镇建设、基础设施建设等方面开展合作，早日取得合作成果。
　　陈清表示，中铁二十二局集团是一家资本雄厚、实力非常强的企业，拥有丰富的投资和建设施工等经验，全区各部门要积极对接，为中铁二十二局做好服务，通过多种模式开展合作，掀开房山经济发展新的一页。
　　区领导刘兵、齐文东出席会议。
